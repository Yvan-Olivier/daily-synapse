# Daily Synapse

**A daily multi-agent AI news intelligence platform.**

Every morning, Daily Synapse automatically aggregates the latest tech and AI news, processes it through a chain of specialized agents, and delivers it in two formats: a curated **HTML email digest** and a **daily audio podcast** generated by text-to-speech.

The idea: stay on top of AI without the noise — read it or listen to it, every day.

---

## What it does

1. **Collects** content from major AI sources (Anthropic, OpenAI, Mistral, Hugging Face, arXiv…)
2. **Summarizes and filters** each article using a cost-aware multi-LLM strategy
3. **Curates** the most relevant content based on a user profile
4. **Delivers** a daily email digest and a 3–5 min audio podcast

---

## Key features

- **Multi-agent orchestration** — a LangGraph pipeline coordinates specialized agents (Scraper, Summarizer, Critic, Curator) with parallel output generation
- **Anti-hallucination** — a dedicated Critic agent verifies every summary against the source before it reaches the user
- **Daily TTS podcast** — a narrative audio briefing synthesized from the day's top stories (Azure AI Speech)
